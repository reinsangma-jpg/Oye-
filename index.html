About This Project

ChessMaster Pro is a comprehensive web application that combines the beauty of 3D chess with practical life-tracking tools.

Features:

  • ✨ Stunning 3D chess gameplay with WebGL
  • 🤖 AI opponent with multiple difficulty levels
  • 📊 Game history and replay system
  • 📅 Chess practice attendance tracker
  • 💰 Personal finance and asset management
  • 📈 Interest calculation and projections
